# DnD Campaign Organizer

A simple place to organize my D&D games. It keeps character sheets, campaign details, useful tools, and reference sources together so I can find what I need and figure things out as each game develops.

The repository root is the main information directory. Each campaign has its own subfolder, such as `Heathcrow`.

## Contributing

Help keep information clear, useful, and easy to find. Prefer small, straightforward additions and avoid adding structure or tools until they are actually needed. Follow YAGNI principles.
